# How to install it

```
git clone --recurse-submodules git@github.com:robjperez/vim.git
mv vim ~/.vim
cp ~/.vim/vimrc ~/.vimrc
```

# How to update submodule
```
git submodule update --remote
```

# Install Vundle plugins
```
vim +PluginInstall +qall
```
